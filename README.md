# CamCV

Motion detection on webcam using openCV Framework.  
Continuous learning differences amount from a previous and current image.  
Trigger action when diff ratio greater than acceptance.  

## Requirements

* Boost lib (v1.78.0)
* OpenCV lib (v4.x)
* C++ compiler (v10.2.1)
